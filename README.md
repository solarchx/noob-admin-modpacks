# List of modpacks for Noob//Admin SMP
If you need the ZIP files, ask someone who uses **PrismLauncher** or the **Modrinth App**. Then, ping **Ven** with said ZIP file uploaded to a file hosting service (can be [Catbox](https://catbox.moe) with a 200 MB file size limit, [Litterbox](https://litterbox.catbox.moe/) with a 1 GB file size limit for up to 3 days, or [file.io](https://www.file.io/) with a 4 GB file size limit for up to 7 days (ven recommended))

Everything in the Side Content directory may not be hosted in the Noob//Admin server and is most likely hosted separately
